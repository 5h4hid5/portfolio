About Me

Hi, I am Shahid from Dhaka, Bangladesh. I like to introduce myself as a full stack web developer. I started my programming study about 3 years ago when I started my undergraduate study by learning Java programming. I have learnt data structures, different types of algorithm and the OOP concepts as my academic course. But about two years ago I found myself interested in learning web programming. Till now I have learned and mastered HTML, CSS, JavaScript, jQuery, Responsive Design, Bootstrap and MySQL.

I have been learning PHP and and Laravel (MVC Framework) for the last ten months and successfully completed a Laravel project (www.rateit.com.bd).

On my pastime I like to read, write blog and design websites and logos.
